RoR2VersionSelector
===============

RoR2 Version Selector. Uses DepotDownloader behind the scene.

- [Download it here](https://github.com/risk-of-thunder/RoR2VersionSelector/releases/download/nightly/binary.zip).
- Extract.
- Open `RoR2VersionSelector.exe`.
- Select the RoR2 version you want to play.

Note: Steam Username and Password are required for DepotDownloader to works correctly. No credentials are stored on disk.
